## Overview

Numina is a group fitness social platform that combines elements of ClassPass, Meetup, and community-focused fitness. This backend provides:

- User authentication and profile management
- Fitness class catalog and discovery
- Matching algorithms (user-to-user, user-to-class)
- Messaging and coordination features
- Ratings and feedback system

## Technology Stack

- **Language**: Kotlin JVM
- **Framework**: Ktor 3.0+
- **Database**: PostgreSQL (SQLite for dev)
- **DI**: Koin
- **Authentication**: JWT tokens
- **ORM**: Exposed
